# monte-car-bg

Генератор hero-фонов с машиной для модельных страниц Monte Tuning.

Вставляешь список машин (по строке, или перетащи `.txt`/`.csv`) → инструмент подбирает окружение под тип каждой (город / трасса / бездорожье / побережье) и рисует кинематографичный фон (`gpt-image-2`) в фирменном стиле. Превью + скачивание (по одной или «Скачать все»).

**Живой инструмент:** https://soleilsoules.github.io/monte-car-bg/

## Токен

Ключ OpenAI вводится прямо на странице и хранится **только в localStorage браузера** — в репозиторий и на сервер не попадает. CORS у OpenAI открыт, поэтому браузер ходит в API напрямую, без бэкенда. На общем компьютере жми «Забыть».
